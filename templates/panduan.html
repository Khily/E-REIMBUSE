Panduan Penggunaan

Petunjuk lengkap menggunakan aplikasi E-Reimburse

1

Buka Menu Reimburse App

Reimburse App adalah sebuah menu apabila anda ingin melakukan input reimburse.

2

Pilih Button " + "

Klik Button " + " untuk menambahkan input reimburse baru.

3

Input File Yang Dibutuhkan

Anda dapat memasukkan file berupa gambar yang diminta pada halaman input, lalu klik tombol " Kirim "

4

History

Anda dapat melihat riwayat reimburse yang sudah pernah di input. Pada menu " History " yang terletak pada halaman dashboard.

5

Catatan Penting

Apabila anda sudah tidak dapat melakukan penginputan maka artinya anda sudah mencapai limit di bulan tersebut.

Selamat menggunakan aplikasi E-Reimburse!
